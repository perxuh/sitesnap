# SiteSnap Product Requirements

## What SiteSnap Is

SiteSnap is a phone-first AI website builder and website manager for local small business owners, especially contractors and service businesses that do not want to learn traditional website tools.

## Target Customers

- Excavation and demolition companies
- Landscapers
- Roofers
- Plumbers
- HVAC companies
- Cleaning businesses
- Other local service businesses

## Core User Flow

1. The business owner opens the app on their phone.
2. They answer a guided set of simple business questions.
3. They add branding details, photos, reviews, and a preferred domain.
4. SiteSnap generates a structured website preview.
5. The owner approves a mock publish flow.
6. The app becomes a dashboard for future edits and lead tracking.
7. The owner later requests updates through an AI prompt, reviews the changes, and approves publishing.

## Main Features

- Guided onboarding questionnaire
- Structured website generation
- Mobile and desktop preview
- Mock publish flow
- Post-publish dashboard
- AI-assisted edit requests with preview-before-publish
- Basic lead capture

## What Makes It Different

- Built for people who do not want to learn web design
- Mobile-first instead of desktop-first
- Uses guided questions instead of open-ended editing
- Keeps AI changes controlled through structured data and approval

## MVP Scope

- Clickable mobile app prototype
- Local state for onboarding and preview flow
- Structured website data model
- Template-based website preview
- Mock publishing
- Dashboard shell
- Mock AI edit flow
- Basic mock leads

## Out Of Scope For Now

- Real domain purchasing
- DNS automation
- Payments
- Multi-user roles
- Advanced CRM
- Appointment booking
- Ad campaign management
- Email marketing
- Complex analytics
